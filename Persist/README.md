
### 공부 방법 및 순서

- 백준과 프로그래머스 병행공부
- 순서
    1. 언어 문법공부
    2. 백준 기본문제
    3. 프로그래머스 고득점 Kit를 통해 자료구조 및 유형별 알고리즘 이론문제
    4. 유형별 알고리즘 이론공부

- 자료구조 및 알고리즘 난이도 및 공부순서
    - 기본 : Queue, Stack, LinkedList, Hashmap, String
    - 중급 : Priority Queue,  DFS, BFS, 완전탐색, Binary search, Recursion
    - 심화 : Graph, DP
