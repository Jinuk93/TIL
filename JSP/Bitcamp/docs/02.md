***Call Back (콜백 메소드)***

**어떤 시점이 되면 JVM에 의해서 호출되는 메소드**

**아래의 세 가지 메소드 모두, 콜백 메소드이다**

- **init() - 처음 한번만 실행된다**
- **doGet - 요청이 있을때 - service()**
- **destroy - 서블릿 죽을 떼 - 한번만 실행된다(어차피.. 끝나지만...)**

- **init() → doGet → destroy 함수를 도는 것 : 주기함수(Life Cycle)**
